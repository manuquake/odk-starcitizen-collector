ODK Star Citizen Collector - Privacy summary

This installer installs a local companion collector for Star Citizen.

The collector reads:

- the configured Star Citizen Game.log file;
- the ScreenShots folder near the configured game installation, if screenshot
  proof uploads are enabled in configuration;
- its own local configuration file.

The collector sends data only to the backend URL configured by the user during
setup. The default ODK backend is:

https://app.odkclan.it

It does not read Star Citizen process memory, does not inject code into the
game, and does not modify Star Citizen files.

The collector token is not bundled with this installer. It is generated by the
backend and stored locally in:

%APPDATA%\odk-control-center\starcitizen-collector.json

For full details, see PRIVACY.md in the public repository:

https://github.com/Manuquake/odk-starcitizen-collector
